```javascript
var input = require('fs').readFileSync('/dev/stdin').toString().trim().split('\n'); 
const n = +input[0];
let dp = [];
dp = new Array(n + 1).fill(0);

dp[1] = 0;
for(let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + 1;
    if(i % 2 === 0) dp[i] = Math.min(dp[i / 2] + 1, dp[i]);
    if(i %3 === 0) dp[i] = Math.min(dp[i / 3] + 1, dp[i]);
}
console.log(dp[n]);
```

DP는 많이 풀어봐야하고 여러가지 다양한 점화식을 접해봐야 한다
