1을 뺀다.

    '''
    f(10) -> f(5) f(3)
    f(5) -> f(2) f(1)
    f(3) -> f(1)
    10 -> 5
    10 -> 3 + 1
    3 -> 1
    5 -> 2 + 1
    '''
