## 10819번 차이를 최대로

### 난이도

실버2

### 문제

N개의 정수로 이루어진 배열 A가 주어진다. 이때, 배열에 들어있는 정수의 순서를 적절히 바꿔서 다음 식의 최댓값을 구하는 프로그램을 작성하시오.

> |A[0] - A[1]| + |A[1] - A[2]| + ... + |A[N-2] - A[N-1]|

### 입력

첫째 줄에 N (3 ≤ N ≤ 8)이 주어진다. 둘째 줄에는 배열 A에 들어있는 정수가 주어진다. 배열에 들어있는 정수는 -100보다 크거나 같고, 100보다 작거나 같다.

### 출력

첫째 줄에 배열에 들어있는 수의 순서를 적절히 바꿔서 얻을 수 있는 식의 최댓값을 출력한다.

### 예제 입력 1 복사

```
6
20 1 15 8 4 10
```

### 예제 출력 1 복사

```
62
```

### 풀이

순열을 이용한 문제이다.
하나의 순열을 뽑아 위 식대로 계산하는 작업을 순열마다 수행 후
그중 가장 큰 값을 찾으면 된다. 파이썬에는 순열을 구하는
라이브러리가 존재하므로 보다 쉽게 가능하다.
js에서는 구현을 해야되는데, 우선 **순열함수(permutations)** 를 만들고
순열함수에서 문제의 식에서 **최대값을 구하는 함수(checkMax)** 를 만들었다.
순열함수에서 만약에 check[i]가 존재시 continue를 시용하여
다음 for문을 수행하게 하였는데, 이유는 중복을 방지 하기 위함이다.

> ex: 10,20,30 이 존재시 10,10,10 / 10,10,20 등을 방지

이후 순열의 결과값은 arr[i]가 selected배열에 담기게 된다.
**permutations(depth+1)** 을 하면서 재귀식으로 순열을 구하고자 하는
배열에 길이만큼 돌수 있게 한다.
이후 다시 루프를 돌기 위해서 check[i]를 false로 만든다.
이처럼 수행하게 되면 순열의 중복이 아닌 결과값이 나오게 된다.
최대값함수는 파이썬과 js와 약간 다르게 설계가 되어 있는데,
파이썬은 for문이 한번더 있고, js는 한번이다. 파이썬은
for i in arr에서 i에 이미 순열이 담겼기에 for문을 한번더 넣어주어,
인덱스 값을 찾아 데이터를 뽑게 하였고, js는 check_arr이라는 인자에
애초에 순열배열 데이터 1개 담기기에 for문을 한번만 사용했다.
check_arr[j]-check_arr[j-1]로 수행하게 되면 문제와 다르게
빼는 방향이 반대가 아닌가 할수 있지만 abs함수로 절대값을 뽑기에
반대로 해도 문제가 없다.

> ex: 50-10 = 40 / 10-50 = abs(-40) => 결과는 같다.

위 설명에도 이해가 잘 되지 않는다면

> ex: arr = [10,20,30] 결과=30

위 예시로 설명을 대입해가면서 수행해보자
