## 10872번 팩토리얼

### 난이도

> 브론즈3

### 문제

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

### 입력

첫째 줄에 정수 N(0 ≤ N ≤ 12)가 주어진다.

### 출력

첫째 줄에 N!을 출력한다.

### 예제 입력 1 복사

```
10
```

### 예제 출력 1 복사

```
3628800
```

### 풀이

작성한 코드는 팩토리얼의 공식이라고 할정도로 대중적인 코드다. (외우는것을 추천)
여기서 유의할점은, n<=1 이나 n==1 이나
같지만, 백준에서 n==1은 런타임에러가 발생한다.
그러므로 n<=1로 설정해주자

팩토리얼 자체의 함수도 파이썬은 제공한다
